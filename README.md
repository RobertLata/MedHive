# MedHive

## Despre MedHive

MedHive este o aplicație mobilă inovatoare, concepută pentru a facilita accesul rapid și ușor la medicamente printr-un sistem de comandă și livrare eficient. 

## Rularea Proiectului

Pentru a rula MedHive pe dispozitivul local, trebuie urmați pașii de mai jos. Aceste instrucțiuni presupun că aveți deja instalat Flutter și Android Studio sau alt IDE pe sistemul dumneavoastră:
1. Navigare în directorul unde s-a clont proiectul.
2. Rularea în terminat sau IDE-ul utilizat a seriei de comenzi `flutter clean && flutter pub get && flutter packages get`
3. Rularea comenzii `flutter run` din terminal sau IDE sau apăsarea butonului run din IDE
4. Selectarea dispozitivului pe care se dorește rularea aplicației

### Pre-rechizite

Asigură-te că ai următoarele instalate:
- Flutter SDK (verifică versiunea necesară în `pubspec.yaml`)
- Android Studio sau orice alt IDE compatibil cu Flutter
- Un emulator, simulator sau un dispozitiv fizic

### Notă despre compatibilitate

Aplicația MedHive este dezvoltată folosind Flutter, ceea ce înseamnă că este compatibilă atât cu Android, cât și cu iOS.
